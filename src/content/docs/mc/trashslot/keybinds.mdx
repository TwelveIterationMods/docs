| Key Binding | Description |
|-------------|-------------|
| <kbd class="kbd">Delete</kbd> | Delete item under mouse cursor |
| <kbd class="kbd">Shift</kbd> + <kbd class="kbd">Delete</kbd> | Delete all items of the same type |
| <kbd class="kbd">T</kbd> | Toggle Slot Visibility. This works in nearly all container screens. |
| <kbd class="kbd text-gray-700">none</kbd> | Toggle Slot Lock. This prevents the slot from being moved around when dragging it. Unbound by default. |
| <kbd class="kbd">Drag & Drop</kbd> | Move the TrashSlot to a different position. |
