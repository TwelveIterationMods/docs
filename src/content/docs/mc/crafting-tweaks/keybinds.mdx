| Key Binding                                     | Description                                                           |
| ----------------------------------------------- | --------------------------------------------------------------------- |
| <kbd class="kbd text-gray-700">none</kbd>       | Rotate the crafting grid.                                             |
| <kbd class="kbd text-gray-700">none</kbd>       | Rotate the crafting grid counter-clockwise.                           |
| <kbd class="kbd text-gray-700">none</kbd>       | Balance the crafting grid.                                            |
| <kbd class="kbd text-gray-700">none</kbd>       | Spread items around the crafting grid.                                |
| <kbd class="kbd text-gray-700">none</kbd>       | Clear items from the crafting grid.                                   |
| <kbd class="kbd text-gray-700">none</kbd>       | Force clear items from the crafting grid, dropping them if necessary. |
| <kbd class="kbd">K</kbd>          | Compress a stack of items using a 2x2 or 3x3 recipe.                  |
| <kbd class="kbd">CTRL + K</kbd>   | Compress one item using a 2x2 or 3x3 recipe.                          |
| <kbd class="kbd">SHIFT + K</kbd>  | Compress all items of a type using a 2x2 or 3x3 recipe.               |
| <kbd class="kbd text-gray-700">none</kbd>       | Decompress a stack of items.                                          |
| <kbd class="kbd text-gray-700">none</kbd>       | Decompress one item.                                                  |
| <kbd class="kbd text-gray-700">none</kbd>       | Decompress all items of a type.                                       |
| <kbd class="kbd">CTRL + TAB</kbd> | Refill the last crafting recipe for one item.                         |
| <kbd class="kbd">TAB</kbd>        | Refill the last crafting recipe for a stack of items.                 |
| <kbd class="kbd text-gray-700">none</kbd>        | Transfer a stack of items to the crafting grid.                 |
